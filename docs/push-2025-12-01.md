## التحقق من الدفع (Git)
- تم تنفيذ `git push -u origin main` بنجاح (بحسب سجل الطرفية).

## تشغيل محلي سريع (Windows PowerShell)
- بدون تكلفة (Stub):
```powershell
npm run dev:mem
```
- مع ردود حقيقية (OpenAI):
```powershell
$env:OPENAI_API_KEY="sk-..."; npm run dev:mem
```
- اختبار سريع للـ API:
```powershell
Invoke-RestMethod -Method Post -Uri http://localhost:3002/api/ai/chat `
  -Body '{"prompt":"مرحبا"}' -ContentType 'application/json'
```

- انقل `agent-core/` و`agent-core/profile.json` معك لأي مشروع/سطح.
- العملاء (ويب/CLI/VS Code) يظلون رُقَق (Thin Clients) ويتصلون فقط بـ `POST /api/ai/chat`.

## خطوات لاحقة اختيارية
- بث تزايدي عبر SSE/JSONL.
- MemoryStore موحّد (SQLite + Embeddings) بطبقة تجريد قابلة للاستبدال.
- طبقة أدوات بروح MCP ثم تغليف MCP لاحقًا.
- عميل CLI بسيط يستهلك نفس الـ API.
